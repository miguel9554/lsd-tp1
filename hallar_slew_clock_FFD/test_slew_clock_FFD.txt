**** Script en NUTMEG para ser llamado por Python ****

** Descripcion: simula los flancos LH y HL del slew de entrada del FFD

.control
* Cargar circuito
source test_slew_clock_FFD.cir

* Configurar el formato con el que se
* guardaran los datos de salida
set noprintheader
set noprintindex
set nobreak

**********************
** Transicion LH *****
**********************
tran 10p 155n 145n

* Guardar resultados
print v(2) > transicion_LH_clock.txt

**********************
** Transicion HL *****
**********************
tran 10p 110n 90n

* Guardar resultados
print v(2) > transicion_HL_clock.txt

.endc
.end
