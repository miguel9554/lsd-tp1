**** Script en NUTMEG para ser llamado por Python ****

** Descripcion: simula los flancos LH y HL del FFD
** al producirse un flanco ascendente de reloj

.control
* Cargar circuito
source test_tabla_clock_to_Q_FFD.cir

* Configurar el formato con el que se
* guardaran los datos de salida
set noprintheader
set noprintindex
set nobreak

**********************
** Transicion LH *****
**********************
tran 1p 155n 145n
*tran 1p 155n 135n

* Guardar resultados
print v(2) v(5) > transicion_LH_FFD.txt

**********************
** Transicion HL *****
**********************
tran 1p 280n 240n
*tran 1p 80n 30n

* Guardar resultados
print v(2) v(5) > transicion_HL_FFD.txt

.endc
.end
