******************************************************
**** Script en NUTMEG para ser llamado por Python ****

** Descripcion: obtiene el rise time y el fall time
** De un inversor balanceado y descargado y
** lo guarda en los archivos:


.control
* Cargar circuito
source test_slew_salida_inversor_cargado.cir

* Configurar el formato con el que se
* guardaran los datos de salida
set noprintheader
set noprintindex
set nobreak

******************
** Fall time *****
******************

* Simular fall-time
tran 0.1p 6n 1.5n

* Guardar resultados
print v(2) > fall_time_salida_inversor_cargado_max.txt
print v(5) > fall_time_salida_inversor_sin_cargar.txt

******************
** Rise time *****
******************

* Simular rise-time
tran 0.1p 7.5n 6n

* Guardar resultados
print v(2) > rise_time_salida_inversor_cargado_max.txt
print v(5) > rise_time_salida_inversor_sin_cargar.txt

.endc
.end
